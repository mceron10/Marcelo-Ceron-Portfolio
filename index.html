Automated Plant Watering System

Embedded Systems Project

Designed and built an automated plant watering system using microcontroller-based control logic to monitor soil moisture levels and regulate water delivery.

System Design & Implementation:

  • Integrated soil moisture sensors to continuously monitor real-time soil conditions
  • Programmed control logic to activate a water pump when moisture dropped below threshold
  • Calibrated sensor readings to prevent overwatering and root damage
  • Implemented delay logic and safety constraints to improve system reliability
  • Tested system performance under varying environmental conditions

This project required understanding of sensor interfacing, analog-to-digital conversion, embedded programming logic, and practical circuit integration. The system successfully automated irrigation while minimizing water waste and improving plant health consistency.
