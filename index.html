Projects

KL43Z Interrupts & SysTick Embedded Systems Project

2025

This project helped me understand how embedded systems respond to real-time inputs using low-level programming. Using Embedded C on the NXP KL43Z microcontroller, I implemented both interrupt-driven and timer-based control systems.

  • Used GPIO interrupts to trigger an Interrupt Service Routine (ISR) from a push button
  • Implemented conditional logic inside the ISR to control LED behavior
  • Configured the SysTick timer to generate precise 0.25-second intervals
  • Worked directly with hardware registers and microcontroller peripherals
  • Improved understanding of real-time systems and efficient event handling

Automated Plant Watering System

(Java & MATLAB)

Designed and developed an automated irrigation system that monitors soil moisture levels and controls water delivery using microcontroller-based logic. This project combined embedded programming in Java with MATLAB for data analysis and system validation.

  • Integrated soil moisture sensors for continuous real-time monitoring
  • Developed control logic in Java to activate a water pump based on threshold values
  • Used MATLAB to analyze sensor data and optimize system performance
  • Calibrated sensors to prevent overwatering and improve plant health
  • Implemented timing delays and safety constraints for reliable operation
  • Tested system under different environmental conditions to ensure stability

This project strengthened my understanding of sensor interfacing, data analysis, and the integration of software with physical hardware systems.
