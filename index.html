KL43Z Interrupts & SysTick Embedded Systems Project

2025

This project was a hands-on way for me to better understand how embedded systems respond to real-world inputs. I worked with the NXP KL43Z microcontroller using Embedded C to explore interrupt-driven programming and timer-based control, and I had a really fun time building and testing the system.

  • Implemented GPIO interrupts where a button (SW3) triggers an Interrupt Service Routine (ISR)
  • Designed ISR logic to check a second button (SW1) and conditionally toggle an LED
  • Configured the SysTick timer to toggle an LED every 0.25 seconds
  • Worked directly with low-level hardware registers and microcontroller peripherals
  • Compared interrupt-based design vs polling for efficiency
